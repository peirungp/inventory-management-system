# Inventory Management System 

Inventory management system is designed for wholesalers working with multiple suppliers and high-volume product catalogs. The system centralizes supplier and product management while reducing manual effort through automated inventory monitoring logic and restocking workflows.

By leveraging Java and a structured MVC architecture, the system provides clear role separation and efficient data organization for administrators, suppliers, and wholesalers.

## Tech Stack
- Java
- Eclipse
- JavaFX with Scene Builder
- Model-View-Controller(MVC)

### Features

- For Wholesalers
  - Real-time inventory overview
  - Low-stock alerts and replenishment workflows
  - Order creation for importing stock from suppliers
- For Suppliers
  - Independent product management
  - Add, update, and remove products
  - Maintain accurate pricing, quantity, and descriptions
- For Administrators
  - Supplier management
  - Add, view, and modify supplier information
  - System-level oversight of suppliers and inventory structure
